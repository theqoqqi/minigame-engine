Мини-движок для игр

Движок основан на jQuery и предоставляет небольшой набор функций для создания простейших игр, используя обычные jQuery-элементы в качестве игровых объектов.

Список возможностей:

  • Создание и управление объектами на игровом поле
  • Базовая физика (проверка столконвений, перемещение с учетом препятствий)
  • Панель состояния, на которой можно отобразить счет и прочую информацию
  • Набор функций для работы с клавиатурой и мышью
  • Прочие утилитные функции для работы с прямоугольниками

Базовый пример

Сперва движок нужно настроить и запустить: 
                    engine.setWorldSize(400, 400);
                    engine.setTps(60);
                    engine.gameLoop();
Затем можно создать объект игрока: 
                    var $player = engine.createElement('player', 180, 180, 40, 40);
                    engine.addElementToWorld($player);
Однако, без стилей игрока не будет видно, поэтому добавим немного стилей: 
                    .player {
                        background-color: #0f74a8;
                    }
И, наконец, отправим игрока куда-нибудь на тот свет: 
                    engine.setUpdateCallback(() => {
                        var position = engine.getElementPosition($player);
                        engine.setElementPosition($player, position.x + 1, position.y - 1);
                    });
Готово! Теперь мы можем увидеть игрока посреди поля, убегающего куда-то далеко-далеко.
